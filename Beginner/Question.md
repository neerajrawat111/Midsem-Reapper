#  Export Multiple Elements Using CommonJS

##  Problem Statement

You have to create the JavaScript elements in the **`index.js`** file.

Create the following:

1. A string named **`studentName`** with any value.  
2. A number named **`studentAge`** with any value.  
3. A function named **`greetStudent`** that returns the string  
   `"Welcome to Advanced Programming!"`.

Your task is to **export all three** using **CommonJS module syntax** as part of an object.
